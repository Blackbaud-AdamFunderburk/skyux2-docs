--logFormat â€” Specifies how to display logging information in the command prompt. To display detailed and expanded compilation logs, use --logFormat expanded. This is the default for most SKY UX CLI commands, and we generally recommend the expanded format for CI environments because it displays each log on its own line. To display summary compilation logs and reuse lines to take up less space in the command prompt, use --logFormat compact. The compact format is the default for skyux serve and skyux build --serve. To display all logs on a single line that is constantly updated, use --logFormat simple. To hide all compilation logs, use --logFormat none.
