The progress indicator component visually represents progress through a series of sequential steps toward a final goal. The progress indicator can represent a series of user actions that are a part of direct workflow, or it can represent passive progress through steps that concern users but that they do not directly act upon.

Specifies whether to display the progress indicator vertically or horizontally. Specifies an observable of SkyProgressIndicatorMessageType that determines the status to display for progress items in the progress indicator. The message stream is a queue of commanding messages to change the state of the progress indicator based on one of the three message types. Specifies the index for the first progress item to make active when the progress indicator loads. Fires when the progress indicator changes the status of a progress item. Indicates that the current step is complete. This completes the active progress item and moves to the next progress item. Indicates that progress should return to the previous item. This moves from the currently active item to the item that precedes it. Indicates that progress is incomplete. This marks all items as incomplete and sets the active item to the first progress item. Specifies the title to display with the progress item. Indicates whether the reset button is disabled. Specifies the progress indicator component to associate with the reset button. Fires when users select the reset button to mark all items as incomplete and set the active item to the first progress item. Specifies the label to display on the navigation button. Specifies the type of navigation button to include with the progress indicator when it is used with a wizard component. Options include finish, next, and previous. Indicates whether the nav button is disabled. Specifies the progress indicator component to associate with the nav button.
