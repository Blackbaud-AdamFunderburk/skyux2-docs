The popover module allows you to display contextual content when users interact with an element on a page. The content can be plain text, HTML, or an Angular component.

Use when

  • You need to give users access to small pieces of contextual content that are not always valuable to see.
  • You need to guide users toward subtle-but-important elements when tasks include complex or unfamiliar capabilities.

Don't use when

  • The content is valuable to see most of the time. Use presistent inline help instead.
  • You need to show a large amount of content. Use a flyout instead.

Positioning

You can specify the position of the popover in relation to the element it is pointing to. The default position is above and centered on the element.

Trigger

To ensure usability on touch devices, trigger user-invoked popovers on click rather than mouseenter.

You can programatically invoke popovers, for example the first time a user visits a page to call out an important interface element that is easily overlooked. However, don't programatically invoke more than one popover on a page or they will lose effectiveness at drawing attention.

Closing

Popovers close when the user clicks on or focuses an element outside the popover.

Responsiveness

If the popover doesn't fit in the viewport in its current position it will automatically try to reposition itself so that it fits. If it does not fit in any position it will convert itself into a modal overlaying the page.

Keep the amount of content in a popover small. The content can be simple text or formatted HTML.

Don't include too much content, form inputs or complex interactions.

You can use small amounts of plain text content.
You can use small amounts of formatted HTML content.
Don't include large amounts of content.

In most cases, use the default “above” placement for Popovers. This minimizes the chances that the Popover will obscure content that a user hasn’t seen yet or will interact with next.

Use other placement options if the trigger for the Popover will be close to the edge of the display. Choose a placement that will obscure as little other content as possible.
