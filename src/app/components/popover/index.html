The popover module allows you to display contextual content when users interact with an element on a page. The content can be plain text, HTML, or an Angular component.

Popovers display small chunks of contextual content that do not always need to be visible. Popovers are appropriate when:

Use when

  • Users select a button or link to display the popover, and the interaction does not change the system state.
  • The popover content directly relates to the user's current task.
  • The popover content does not require scrolling on a mobile device.

Don't use when

If the content is valuable to see most of the time, use presistent inline help instead. If the amount of content requires scrolling on a mobile device, use a flyout instead.
Interaction

To ensure usability on touch devices, you should generally trigger popovers on click rather than mouseenter.
